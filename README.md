
**宠物领养系统**
## 一、 项目介绍
      管理员：首页、个人中心、宠物分类管理、宠物用品、宠物商店管理、宠物领养管理、用户管理

      用户： 首页、个人中心、宠物领养管理、宠物寄存管理、宠物挂失管理

## 二、 运行环境
      开发语言：Java

      开发工具：IDEA，Eclipse,Myeclipse都可以。推荐IDEA

      JDK版本：1.8

      数据库: MySQL5.7

      Tomcat环境：Tomcat 7.x,8.x,9.x版本均可

      项目管理：Maven

## 三、 项目技术
      后端：SSM(Spring+SpringMVC+Mybatis)

      前端：ElementUI+Vue
